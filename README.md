# MyTextTXX5（学习交流自用）
tbs x5 webview 打开office 文件(毕竟IOS webview自带这种功能，android 一句实现不了 很尴尬！)
原理：实际是把问价下载之后 用webview 打开（原生webview 好像不具备这个）
优化：处理一下 打开方式  避免每次打开都需要加载好久！

内部原理 可以看腾讯X5内核 官方文档 https://x5.tencent.com/tbs/guide/sdkInit.html
